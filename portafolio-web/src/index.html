Sobre Mí

Soy Kevin Sánchez, un ingeniero de software dedicado a crear aplicaciones eficientes y escalables. Tengo una sólida experiencia en desarrollo full-stack, con un enfoque en tecnologías modernas como Angular, Ionic y Node.js. También tengo un conocimiento práctico de Python (Django) y PHP para el desarrollo del backend. Mi experiencia incluye la gestión de bases de datos relacionales (MySQL, PostgreSQL) y un fuerte manejo de control de versiones con Git.

Mi pasión es el aprendizaje continuo y la aplicación de nuevas tecnologías para resolver problemas complejos. Me comunico fluidamente en español e inglés (B2 certificado), lo que me permite colaborar de manera efectiva en equipos internacionales. Busco un entorno desafiante donde pueda aplicar mis habilidades y crecer profesionalmente.
